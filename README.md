#MicroMoney

和室友记账使用的简单的Web程序.

依赖的技术有

+ MicroMan框架(-_-# 你没看错,就是Framework里面写的那个文件,我人生第一个MVC框架...)
+ AdminLTE模版,正在接入.
+ WebSocket  (如果要加入和室友聊天的话....)
